# RestaurantOrderSystem

Open this project in IntelliJ as a Java project (use Java 21 / LTS).

Structure:
- src/main/java/restaurant/... (Java source)
- report/Report.md
- UML/restaurant_uml.png (simple UML diagram)

Build & Run:
- Compile with `javac` and run Main, or open in IntelliJ and run Main.main.

Notes on features used:
- Sealed classes, records, varargs, LVTI (`var` can be used inside methods), default/static/private interface methods, lambdas, method references, switch expressions could be extended where appropriate.
