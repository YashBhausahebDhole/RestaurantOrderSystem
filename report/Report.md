# Restaurant Order Management System - Report

## Introduction
A simple Java 21 application demonstrating fundamental and advanced OOP features as required by the OOP1 assignment.

## User stories completed
- Add, remove, view menu items.
- Place an order with multiple dishes.
- Calculate total price and generate immutable receipt.
- Input validation via exceptions.
- Filter items with predicates (lambda).

## Evaluation
The implementation meets the assignment brief; fundamentals and advanced features are demonstrated.
